# iOS 26 Design System (Dark Mode)

## 1. Color Palette (Consolidated)

| Category | Token Name | Value | Description |
| :--- | :--- | :--- | :--- |
| **Neutral** | `--background-base` | `#000000` | Deepest background layer. |
| | `--surface-primary` | `#0A0A0A` | Standard card surface. |
| | `--surface-glass` | `rgba(255, 255, 255, 0.08)` | Standard glassmorphism layer. |
| | `--surface-glass-heavy` | `rgba(255, 255, 255, 0.15)` | Denser glass for overlays. |
| **Brand** | `--brand-primary` | `#10B981` | Emerald 500 (Core actions). |
| | `--brand-secondary` | `#6366F1` | Indigo 500 (AI/Accent features). |
| **Typography** | `--text-primary` | `#FFFFFF` | Primary headings/body. |
| | `--text-secondary` | `rgba(255, 255, 255, 0.4)` | Subtitles/Labels. |
| | `--text-tertiary` | `rgba(255, 255, 255, 0.2)` | Placeholders/Dividers. |
| **Semantic** | `--semantic-danger` | `#FB7185` | Rose 400 (Warnings/Expenses). |
| | `--semantic-success` | `#10B981` | Emerald 500 (Income/Savings). |
